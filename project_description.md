# Image Lab - Project Overview

**Image Lab** is a high-performance web application designed for seamless image management and creative workflows. Built with a focus on speed, security, and aesthetics, this platform empowers users to organize, upload, and interact with images in a professional environment.

### Core Technology Stack

*   **Framework & Language:** The application is built on **Next.js 16** (utilizing React 19) and **TypeScript**. This ensures the platform is fast, SEO-friendly, and maintainable with strict type safety.
*   **Database & ORM:** For data persistence, the project uses **Neon Serverless Postgres** paired with **Prisma ORM**. This combination provides a scalable and reliable way to manage user accounts, folder structures, and image metadata.
*   **Authentication:** Secure user access is handled by **Better Auth**, providing a modern and robust authentication layer to protect user privacy and data.
*   **Image Management:** Cloud-based image hosting and transformations are powered by **Cloudinary**. This allows for lightning-fast image delivery and automatic optimization for any device.
*   **Styling & UI:** The interface is crafted with **Tailwind CSS 4** and **Shadcn UI** components, ensuring a sleek, responsive, and accessible design. **Framer Motion** is integrated to provide premium micro-animations that make the user experience feel alive.
*   **State Management:** **Zustand** is utilized for lightweight and efficient client-side state management, keeping the UI in sync with the backend.
*   **AI Integration:** With **OpenAI** included in the stack, the platform is ready for advanced features such as AI-driven image generation or automated tagging.

### Summary
Image Lab represents a modern approach to web development, combining cutting-edge tools to create a platform that is not only functional but also visually stunning and highly performant.
